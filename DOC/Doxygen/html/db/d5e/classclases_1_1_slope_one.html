◆ computarDesviacion()

controladores.SlopeOne.computarDesviacion ( )
private

Crear el mapa de desviaciones.

Cogemos la diferencia entre cada usuario que tiene valoración para dos items i, j. Añadimos la desviación al mapa de diff y añadimos a la frecuencia de usuarios que han valorado i y j.

PSEUDOCÓDIGO: for every item i for every other item j for every user u expressing preference for both i and j add the difference in u’s preference for i and j to an average

Precondición
Se han inicializado las variables u y us a un usuario y un conjunto de usuarios respectivamente.
Postcondición
Se han calculado las desviaciones entre todos los items que un mismo usuario haya valorado.

Definición en la línea 178 del archivo SlopeOne.java.

◆ computarPredicciones()

controladores.SlopeOne.computarPredicciones ( )
private

Calcular las predicciones para el usuario u.

A partir del mapa de desviación y frequencia podemos calcular las valoraciones que hara el usuario. Cogeremos las diferencias de todo j e i donde i ha sido valorado por nuestro usuario, y la añadimos a la mediana de valoraciones de nuestro usuario la podemos predecir el valor que le dara nuestro usuario al item j.

PSEUDOCÓDIGO: for every item i the user u expresses no preference for for every item j that user u expresses a preference for find the average preference difference between j and i add this diff to u’s preference value for j return the top items, ranked by these averages

Precondición
Se han inicializado las variable u, us, diff y freq de tal manera que podamos computar las predicciones. Para inicializar a diff y freq se tiene que llamar a computarPredicciones.
Postcondición
Se ha inicializado el mapa de predicciones de manera que este contiene todas las predicciones que podemos hacer de un usuario base.
