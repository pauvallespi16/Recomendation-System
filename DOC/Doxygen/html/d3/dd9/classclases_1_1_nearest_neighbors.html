Descripción detallada

Implementación del algoritmo nearest neighbors, usando content based filtering.

El algoritmo Nearest Neighbors tiene en cuenta la distancia de cada item que ha valorado el usuario con el resto de items que existen, para así recomendarle los items mas similares. Ademas, también se tiene en cuenta la valoración que le ha dado el usuario a sus items.

◆ prediccionValoracionItem()

controladores.NearestNeighbors.prediccionValoracionItem ( Item  item)

Calcula las predicciones de valoraciones para un item que el usuario no ha valorado.

FORMULA: recom(u,i) = ∑j(recom(u,j) * (1-dist (i,j))) Donde u es el usuario, i es el item del cual queremos predecir la valoración y j es cada uno de los items que ha valorado el usuario u.

Precondición
Se ha inicializado la variable usuario a un usuario y la variable item a un item.
Postcondición
Se ha devuelto la predicción de la valoración para el item dado.
Parámetros
usuarioUsuario al cual hay que recomendar items.
itemItem del usuario.
Devuelve
Devuelve la predicción de la valoración para el item dado.
